# bughunt-cli

This command line tool supports bughunting projects.

## install

```shell
git clone git@github.com:howmuch515/bughunt-cli.git
cd bughunt-cli
./install.sh
```

## usage

```shell

bughunt init <myproject>

```

The `myproject` directory structure will be as follows.

```md

myproject/
├── memo.md
├── scan/
├── report/
└── dict/

```
